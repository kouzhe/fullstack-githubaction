# CareerMate AI Landing Page

这是一个使用纯 HTML、CSS 和 JavaScript 构建的 Landing Page。

## 项目结构

```
zeplin-ai-coding/
├── index.html          # 主HTML文件
├── styles.css          # 样式文件
├── script.js           # JavaScript文件
├── assets/             # 资源文件夹
│   ├── logo-icon.png   # Logo图标
│   └── logo-text.png   # Logo文字
└── README.md           # 说明文档
```

## 如何使用

### 1. 保存Logo图片

请将你提供的两张logo图片保存到 `assets` 文件夹中：
- 星形图标 → 保存为 `logo-icon.png`
- 文字logo → 保存为 `logo-text.png`

### 2. 打开页面

直接双击 `index.html` 文件，或者在浏览器中打开它。

### 3. 使用本地服务器（推荐）

为了获得最佳体验，建议使用本地服务器：

```bash
# 使用 Python 3
python3 -m http.server 8000

# 使用 Python 2
python -m SimpleHTTPServer 8000

# 或者使用 Node.js 的 http-server
npx http-server -p 8000
```

然后在浏览器中访问：`http://localhost:8000`
